HEXO建站要点

切换主题

比如要切换minos主题

打开主题的仓库,复制仓库地址git@github.com:ppoffice/hexo-theme-minos.git

在文件根目录打开终端,输入git clone git@github.com:ppoffice/hexo-theme-minos.git themes/minos克隆仓库内容

在_config.yml文件夹中找到theme,更换成theme: minos

然后hexo clean清除上一次缓存,这时可能会报错,提示你有些依赖没有安装

如ERROR Package lodash is not installed.

这时候需要npm i lodash将依赖包安装完毕,以此类推

安装完依赖后hexo clean

hexo g生成部署内容

hexo s在本地预览

hexo d部署

安装hexo-renderer-sass时报错

需要切换镜像

npm install -g cnpm --registry=https://registry.npm.taobao.org

npm config set registry https://registry.npm.taobao.org

cnpm install hexo-renderer-sass --save
